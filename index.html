MTS Stock Shortage System

This is an unofficial Micheldever website that I created to solve a problem I have noticed while working for their company. When it's time to pick tyres for the afternoon vans, we may have to report short stock to the stock operatives. However, we never know whether the stock was actually there or not. This then leads the warehouse operatives to then search for the tyre again not knowing if the stock operatives found it or not. I created this in order for the stock operatives to be able to tell us which tyres are actually short instead of us questioning whether it has been previously looked for.
